# withbuddy

> AI-powered buddy app — TypeScript + React frontend with smart automation integrations
>
> ## 🚀 Overview
>
> **withbuddy** is an intelligent companion app that leverages AI to help users stay organised, automate repetitive tasks, and get smart contextual assistance. Built with a modern React/TypeScript stack and a clean, responsive UI.
>
> ## ✨ Features
>
> - 🤖 AI-driven smart suggestions and task assistance
> - - 📋 Task and goal tracking with intelligent reminders
>   - - ⚡ Real-time updates with instant UI feedback
>     - - 🎨 Clean, responsive UI built with shadcn/ui & Tailwind CSS
>       - - 🔗 Automation integrations for productivity workflows
>         - - 📱 Mobile-friendly design
>          
>           - ## 🛠️ Tech Stack
>          
>           - | Layer | Technology |
>           - |-------|-----------|
>           - | Frontend | React, TypeScript, Vite, Tailwind CSS, shadcn/ui |
> | Build Tool | Vite |
> | Styling | Tailwind CSS |
> | Components | shadcn/ui |
> | Deployment | Vercel |
>
> ## 📦 Getting Started
>
> ### Prerequisites
>
> - Node.js 18+ and npm (or bun)
>
> - ### Installation
>
> - ```bash
>   # Clone the repository
>   git clone https://github.com/clutxhOP/withbuddy.git
>
>   # Navigate to project directory
>   cd withbuddy
>
>   # Install dependencies
>   npm install
>
>   # Start development server
>   npm run dev
>   ```
>
